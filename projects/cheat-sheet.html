Git

Handy tips

Show all the changes for a specific file:
git log -p  <file>
Discard all the changes in the working directory:
git reset --hard HEAD
Show diff for tracked files
git diff

Ruby:

Useful Methods:

each(array):
array.each{|element| puts element}
each(hash):
hash.each{|value, key| puts "#{key} => #{value}"}
map:

Returns an array of whatever is run in the block

array.map{|n| n * 2 }
select:

Returns an array for the elements that meet the criteria of whats in the block

array.select{|n| n.even? }

Other Notes:

All methods that end in ? should return a boolean

All returns are implicit if it is the last line in the method

JavaScript

Useful things to remember

for in
for(var i in object){
    console.log(object[i]);
}
forEach()
[1,2,3,4].forEach(function(x){ console.log(x);})
filter
array.filter(function(x){ return x > 3});
constructor functions for JS:
function Animal(type,name,size){
    this.name = name;
    this.type = type;
    this.size = size;
}

Other Notes:

JavaScript returns are not implicit. You must explicitly return a value
